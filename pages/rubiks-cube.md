## Rubiks Cube Simulator

- C++, Direct3D 11, PhysX

This is a personal project I began to test my new knowledge of 3D application programming.

##### Implemented features:
 - Mouse picking input
 - Auto scramble

##### Planned features:
 - Chamfered edges using a geometry shader
 - Higher fidelity graphics
