## Rubiks Cube Simulator
###### 2016

- C++, Direct3D 11, PhysX

This is a personal project I began to test my new knowledge of 3D application programming.

##### Implemented features:
 - Mouse picking input
 - Auto scramble
 - Smooth rotation animation
 - Input queuing
