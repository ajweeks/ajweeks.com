This is a low poly gun case made using the PBR workflow. (Base colour, roughness, normal, and metalness) This was my first time baking high poly information onto a lower poly game-ready prop.

The screenshots below (excluding the wireframe comparison) were taken using Unreal Engine 4.

Modeling was done in Autodesk's 3Ds Max 2017, and all texturing was done in Adobe Photoshop CC 2017.
