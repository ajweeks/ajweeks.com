This was a three month long project, the goal of which was to have a fully game-ready, realistic prop made using the PBR workflow.

The screenshots above (excluding the wireframe comparison) were taken using Unreal Engine 4.

Modeling was done in Autodesk's 3Ds Max 2017, and all texturing was done in Adobe Photoshop CC 2017.
